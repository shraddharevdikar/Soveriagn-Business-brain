# Sovereign Business Brain

## AI-Powered Autonomous Business Operating System

Sovereign Business Brain is a futuristic AI SaaS platform designed to automate business operations, marketing, lead management, customer engagement, analytics, and AI-driven workflows from one unified ecosystem.

The platform combines:

* AI Agents
* CRM
* Marketing Automation
* AI Analytics
* WhatsApp Automation
* Knowledge Base
* AI Workflows
* Ads Automation
* Voice AI
* Multi-user SaaS Infrastructure

into a single intelligent business operating system.

# Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/sovereign-business-brain.git
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

## Environment Variables

Create `.env` file:

```env
OPENAI_API_KEY=
PINECONE_API_KEY=
DATABASE_URL=
JWT_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```
